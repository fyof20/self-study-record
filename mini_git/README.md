# Mini-Git

> 学习 [MIT](https://github.com/MrBeanCpp/Mit) 项目，从零实现一个迷你 Git

## 学习来源

- **原项目**: [MrBeanCpp/MIT](https://github.com/MrBeanCpp/Mit)
- **课程大纲**: [Tutorial/mit课程大纲.md](../Tutorial/mit课程大纲.md)

## 学习目的

通过复刻 MIT 项目，深入理解 Git 的内部原理：

- 内容寻址存储
- 对象模型（Blob / Tree / Commit）
- 暂存区机制
- 分支管理

## 项目结构

```
mini_git/
├── Cargo.toml
└── src/
    ├── main.rs
    ├── cli.rs
    ├── commands/
    ├── models/
    └── utils/
```
